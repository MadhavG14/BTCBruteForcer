# BTCBruteForcer
This tool can be used to brute force btc wallet 12 word seed phrase 


First install mnemonic bip32 ecdsa requests::
 <b> pip install mnemonic bip32 ecdsa requests </b>

Final command:: 

<b> python bip39_balance_checker.py </b>
